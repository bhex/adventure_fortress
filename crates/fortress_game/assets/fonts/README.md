# Fonts

`DejaVuSansMono.ttf` — DejaVu fonts, v2.37. Embedded at compile time
(`include_bytes!` in `main.rs`) to replace Bevy's limited default font.

License: DejaVu fonts are released under a free license (Bitstream Vera
